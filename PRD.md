# Prompt Hub — 产品需求文档（PRD）

**版本**：v2.0  
**日期**：2026-06-22  
**参考竞品**：[youmind.com/gpt-image-2-prompts](https://youmind.com/gpt-image-2-prompts) · [prompts.sorry.ink](https://prompts.sorry.ink)  
**数据来源**：Image-Prompts（prompts.sorry.ink API），共 956 条有效提示词

---

## 一、产品定位

Prompt Hub 是一个 **AI 提示词发现与复制平台**，帮助用户浏览、搜索、复制高质量 GPT Image 2 提示词，并直接在 AI 图像生成平台中使用。

**核心价值**：降低 AI 绘图提示词门槛，让用户"复制即使用"。

## 三、用户需求分析

| 用户角色 | 核心需求 | 使用场景 |
|----------|----------|----------|
| **AI 绘图爱好者** | 快速找到高质量提示词、一键复制 | 浏览首页推荐 → 点击感兴趣的分类 → 复制提示词去生成 |
| **设计师/创作者** | 按场景（海报/社媒/电商）查找、参考案例 | 搜索 "poster" → 查看详情页的完整提示词 → 替换参数后使用 |
| **新手用户** | 了解如何使用提示词、快速上手 | 首页 → 查看 "How to use" → 复制示例提示词 |
| **贡献者** | 提交自己的提示词、获得曝光 | 注册 → 提交提示词 → 在列表中被展示 |

## 四、功能需求

### 4.1 首页（Homepage）

| 模块 | 功能点 | 优先级 |
|------|--------|----------|
| **Hero 区** | 主标题 + 副标题 + 搜索框（支持回车搜索）| P0 |
| **实时统计** | 展示：总提示词数 / 分类数 / 支持模型数 | P1 |
| **分类入口** | 10 个分类卡片（图标 + 名称 + 数量），点击进入分类页 | P0 |
| **Trending 榜单** | 按浏览量排序，展示 Top 8，卡片含图片+标题+浏览量 | P0 |
| **Latest 最新** | 按时间倒序，展示最新 8 条 | P0 |
| **Featured 精选** | 编辑精选提示词，展示 4-8 条 | P1 |
| **如何使用** | 3 步引导：搜索 → 复制 → 生成 | P1 |
| **Footer** | 版权信息 + GitHub 链接 + 数据来源说明 | P0 |

### 4.2 分类页（Category Page）

| 模块 | 功能点 | 优先级 |
|------|--------|----------|
| **分类标签栏** | 横向滚动标签，当前分类高亮，支持"全部" | P0 |
| **排序切换** | Latest（最新）/ Trending（热门）/ Most Liked（最多点赞）| P1 |
| **提示词网格** | 响应式网格（2→3→4→5列），每张卡片含：图片、标题、分类标签、浏览量、点赞数 | P0 |
| **图片懒加载** | 图片进入视口后才加载，提升性能 | P1 |
| **加载动效** | 图片加载中显示 shimmer 骨架屏 | P0 |
| **空状态** | 该分类暂无提示词时的友好提示 | P1 |
| **分页 / 加载更多** | 每次加载 20 条，点击"加载更多" | P1 |

### 4.3 搜索页（Search Page）

| 模块 | 功能点 | 优先级 |
|------|--------|----------|
| **搜索框** | 支持回车搜索，展示当前搜索词 | P0 |
| **结果统计** | "找到 X 条结果" | P0 |
| **分类筛选** | 按分类筛选搜索结果 | P1 |
| **排序** | Latest / Trending / Most Liked | P1 |
| **搜索结果为空** | 友好提示 + 推荐热门搜索词 | P1 |

### 4.4 详情页（Detail Page）

| 模块 | 功能点 | 优先级 |
|------|--------|----------|
| **面包屑导航** | Home > 分类 > 提示词标题 | P1 |
| **主图展示** | 大图展示（16:9 或原比例），支持点击放大查看 | P0 |
| **图片画廊** | 若有多张生成结果图，展示缩略图列表，点击切换 | P1 |
| **标题** | 中英文标题展示 | P0 |
| **Meta 信息** | 分类标签、贡献者、发布时间、浏览量、点赞数 | P0 |
| **提示词文本** | 完整提示词展示（等宽字体），支持一键复制（复制后 toast 提示）| P0 |
| **参数占位符渲染** | 若提示词含 `{argument name="xxx" default="yyy"}`，渲染为可编辑输入框，复制时自动替换 | P1 |
| **一键复制（智能）** | 若含参数占位符，复制时弹窗让用户填写参数，再复制最终结果 | P1 |
| **Try it now** | 按钮跳转至 AI 图像生成平台（YouMind / ChatGPT 等）| P0 |
| **相关推荐** | 同分类下的其他提示词，展示 4-8 条 | P1 |
| **分享按钮** | 复制链接 / 分享到 Twitter | P2 |

### 4.5 全局功能

| 功能点 | 说明 | 优先级 |
|--------|------|----------|
| **深色模式** | 固定深色（当前已实现），后续可加浅色模式切换 | P2 |
| **响应式** | 完美适配手机（≥320px）/ 平板 / 桌面 | P0 |
| **SEO** | 每张页面独立 title + description + OG 标签 | P1 |
| **性能** | 静态导出（SSG）+ 图片懒加载 + 分页加载 | P0 |
