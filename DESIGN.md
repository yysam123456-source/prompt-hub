## 一、设计语言

### 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（紫）| `#7C77DD` → `#534AB7` | 渐变主色，按钮/高亮 |
| 辅色（粉）| `#D4537E` → `#993556` | CTA 按钮强调色 |
| 背景深 | `#09090B` (zinc-950) | 页面底色 |
| 背景卡 | `#18181B` (zinc-900) | 卡片底色 |
| 边框 | `#27272A` (zinc-800) | 默认边框 |
| 边框高亮 | `rgba(124,119,221,0.4)` | hover/active 状态 |
| 主文本 | `#FAFAFA` (zinc-100) | 标题/正文 |
| 次文本 | `#A1A1AA` (zinc-400) | 辅助信息 |
| 辅助文本 | `#71717A` (zinc-500) | 占位符/说明 |

### 圆角规范

| 元素 | 圆角 |
|------|------|
| 卡片/图片 | `16px` (`rounded-2xl`) |
| 按钮/标签 | `12px` (`rounded-xl`) |
| 胶囊标签 | `999px` (`rounded-full`) |
| 输入框 | `12px` (`rounded-xl`) |

### 字体规范

| 用途 | 大小 | 粗细 |
|------|------|------|
| 页面标题 | `36-48px` | 700 |
| 区块标题 | `24px` | 700 |
| 卡片标题 | `15px` | 500 |
| 正文 | `14px` | 400 |
| 辅助文本 | `13px` | 400 |
| 标签/元信息 | `12px` | 400 |

## 七、动效规范

| 动效 | 技术实现 | 时长 |
|------|----------|------|
| 页面进入 | `framer-motion` `opacity` + `y:30` | 0.6s |
| 卡片 hover | `scale: 1.03` + `border-color` 过渡 | 0.2s |
| 图片加载 | `shimmer` 骨架屏 (CSS `@keyframes`) | 循环 |
| 图片淡入 | `opacity: 0→1` (`transition: 0.7s`) | 0.7s |
| 复制成功 | toast 弹出 → 2s 后消失 | 0.3s |
| 页面切换 | `framer-motion` `AnimatePresence` | 0.4s |
